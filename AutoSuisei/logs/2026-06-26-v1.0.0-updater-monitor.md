# v1.0.0 — monitor barcode + version UI + GitHub auto-updater

- **Date/Time:** 2026-06-26 18:45 (วันศุกร์)
- **Idea Ref:** [[ideas/v1.0.0-release-and-autoupdate]] (+ [[ideas/barcode-qr-serial]] สำหรับ monitor)
- **Changes Made:**
  - **Version 1.0.0 (single source)**: root `CMakeLists.txt` `VERSION 1.0.0` → inject C++ ผ่าน `APP_VERSION`
    (`src/gui/CMakeLists.txt`); sync `installer/AutoSuisei.iss` + `scripts/make_user_guide_pdf.py`
  - **Monitor barcode-first CN serial** (`scripts/bulk_extract.py` + `ocr_worker.py`): `_reformat_cn`
    (dashless → `CN-…-A00`), monitor branch ใน `_pick_serial` (เลือก CN, ทิ้ง Service Tag/A00),
    **rotation sweep** (`_read_barcode_rotated`: upscale 2x + ±3..±12° สำหรับ 1D จอเอียง). วัด **60% decode**
  - **Version UI ซ้ายล่าง + ปุ่ม Check** (`src/gui/SidebarNav.{h,cpp}`: footer `v1.0.0` + "Check for updates"
    → signal `checkUpdateRequested`)
  - **Auto-updater** (ใหม่ `src/gui/Updater.{h,cpp}` + `VersionCompare.h`, link `Qt6::Network`):
    GET releases/latest (tokenless) → semver compare → ดาวน์โหลด `update-package.zip` (fallback installer)
    → `apply-update.bat` (extract→xcopy→relaunch, elevate UAC) → quit. wire ใน `MainWindow.cpp` (dialogs)
  - **Build**: `scripts/make_update_package.ps1` (ใหม่: bundle ลบ `python/` → patch ~30-50MB);
    `.gitignore` เพิ่ม `!AutoSuisei/logs/` (vault logs commit ได้); installer/bundle → 1.0.0
  - **Tests**: GTest `VersionCompare` 9 ตัว (semver) → **118 tests**; self-check monitor (20.jpg = CN-07C2R4-72872-33D-CVTM)
- **Technical Notes:**
  - **Monitor barcode มี CN serial จริง** (ไม่ใช่แค่ Service Tag อย่างที่เดาตอนแรก) — Data Matrix encode CN **ไม่มี dash**
    (`CN0YKH877287273RFUCIA00`) → `_reformat_cn` ใส่ dash ตาม segment CN-6-5-3-4-(A00)?. บางจอเป็น **1D Code128**
    (เช่น 20.jpg) ที่ raw/cv2 อ่านไม่ออก — ต้อง **upscale 2x + ลองหลายมุม** (เจอ +6°). มี Service Tag 7-char ปน → validate CN เท่านั้น
  - **updater tokenless ได้** เพราะ repo public — releases/latest + download asset ไม่ต้อง auth; **push/release เท่านั้นที่ต้อง gh login** (Phase 2b)
  - **partial update ตัด python/ 440MB**: 90% ของ update จริง = scripts/exe; ถ้าเพิ่ม python dep → release ไม่ใส่ update-package → updater fallback full installer
  - **`.gitignore` ไม่รองรับ inline comment** (`#` ต้องขึ้นบรรทัดใหม่) — เคยพลาดทำให้ `!AutoSuisei/logs/` ไม่ทำงาน
  - **apply-update ต้อง elevate (UAC)** เพราะ Program Files protected; updater ใช้ `ShellExecuteW verb=runas`
  - **ระวัง**: auto-updater **ทดสอบ E2E ไม่ได้จน 2b** (ต้องมี release จริง); ตอนนี้ unit (semver) + compile ผ่านเท่านั้น
  - **pushed + released v1.0.0** ✅ — main push (13 commits) + tag + `gh release create` พร้อม assets
    (installer 118MB + update-package.zip 72MB). ทดสอบ API: releases/latest คืน tag v1.0.0 + asset ครบ
    (updater data-path ผ่าน; download-apply เต็มจะทดสอบตอน v1.0.1). gotcha: `GITHUB_TOKEN` env (invalid)
    override keyring login → ต้อง `unset GITHUB_TOKEN` ก่อน push/gh

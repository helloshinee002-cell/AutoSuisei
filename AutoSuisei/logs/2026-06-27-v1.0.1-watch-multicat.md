# v1.0.1 — Folder Watch multi-category (best-of-N)

- **Date/Time:** 2026-06-27 00:10 (เริ่ม 2026-06-26 ดึก)
- **Idea Ref:** ผู้ใช้สั่งตรง — "Folder Watch ติ๊กเลือกได้ว่าจะอ่านหมวดไหน หรือติ๊กทั้งหมด" + bump v1.0.1 push เพื่อเทสต์ in-app updater
- **Changes Made:**
  - **`scripts/ocr_worker.py`** — best-of-N: `--category` รับ CSV (`pc,monitor,accessory,donate`); แยก body ต่อภาพเป็น `process_one(engine,img,meta,category)` (โค้ดเดิม verbatim — barcode-first + donate/monitor fusion); เพิ่ม `process_best()` รันทุกหมวดที่ติ๊ก แล้วเลือกผลดีสุดด้วย `_score = (has_serial, serial_is_barcode, has_no, mean_conf)`; 1 หมวด = ไม่มี overhead. เพิ่ม `--selfcheck` (assert ranking, ไม่พึ่ง OCR)
  - **`src/gui/WatchTab.{h,cpp}`** — เพิ่ม 4 × `QCheckBox` (PC&Laptop/Monitor/Accessory/Donate, default ติ๊กครบ); `selectedCategoryFlags()` → ส่ง `--category=<csv>` ตอน start worker; `updateWatchEnabled()` กันกดเมื่อไม่มีโฟลเดอร์/ไม่ติ๊ก; ล็อก checkbox ระหว่าง watching (worker อ่านหมวดตอน start)
  - **bump → 1.0.1** (3 จุด single-source): `CMakeLists.txt` `project(VERSION)`, `installer/AutoSuisei.iss` `MyAppVersion`, `scripts/make_user_guide_pdf.py` (2 strings)
  - build release → bundle `20260627-000039` → `AutoSuisei-Setup-1.0.1.exe` (120 MB) + `update-package.zip` (76 MB) → GitHub release **v1.0.1**
- **Technical Notes:**
  - 🔑 **1 ภาพ = ของจริง 1 ชิ้น** แต่ละหมวด pipeline คนละแบบ → best-of-N รันทุกหมวดแล้วเลือก ไม่ใช่ classify ก่อน. ค่าใช้จ่าย ~N× ต่อภาพ (donate=Thai Tesseract+digit model หนักสุด) แต่ Watch เป็น human-paced ทีละรูป รับได้
  - smoke (desktop image): per-category breakdown → pc=barcode `4TVQZL2`, monitor=serial ว่าง conf 0.98 → best เลือก **pc (barcode)** ถูกต้อง (has_serial+barcode ชนะ conf). ยืนยันทั้ง 4 branch รันไม่ crash
  - **`serial_is_barcode` tiebreak อยู่เหนือ conf** เพราะบาร์โค้ด decode เชื่อได้กว่า OCR text — กันกรณี 2 หมวดได้ serial เท่ากันแต่หมวดที่ไม่มีบาร์โค้ด conf บังเอิญสูงกว่า
  - **WatchTab ไม่ต้องใช้ enum `AssetCategory` ของ OcrTab** — เก็บคู่ (label, flag) เป็น `kCats[]` local ใน .cpp, index ตรงกับ `catChecks_`
  - **update-package.zip ใช้ได้กับ v1.0.1** — แก้เฉพาะ logic Python (ocr_worker) ไม่เพิ่ม pip dep ใหม่ → ไม่ต้องบังคับ full-installer (ตามกฎใน `make_update_package.ps1`)
  - updater จับ asset ชื่อ **`update-package.zip` เป๊ะ** (`Updater.cpp:78`); tag `v1.0.1` > `1.0.0` → `VersionCompare` ชี้ใหม่กว่า → เครื่องที่ลง v1.0.0 จะเจอ

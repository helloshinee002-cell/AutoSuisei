# v1.0.3 — Review image-first layout + Category columns + branding

- **Date/Time:** 2026-06-27 15:20
- **Idea Ref:** ผู้ใช้สั่งตรง (6 ข้อ) — ต่อจาก [[logs/2026-06-27-v1.0.2-watch-category-cn]]
- **Changes Made:**
  - **A. Review layout** — ย้าย header ทั้งหมด (title+subtitle+Save/Clear+Folder+Verified/progress) ลง**ล่างสุด**, `split` (ตาราง+รูป) **ชิดขอบบน** (`root` top margin 0) → รูปใหญ่สุด
  - **B. Review table: Date → Category** — `categoryLabel(sourceInfos_[i].category)`; + **OcrTab** เซ็ต `info.category = categoryFlag(bulkCategory_)` ให้แถวจาก OCR tab มีค่าด้วย (Watch เซ็ตอยู่แล้ว)
  - **C. Review form: ลบช่อง Batch** (เก็บ Date)
  - **D. Watch table: ลบคอลัมน์ Date** (`kCols` 6→5)
  - **E. Installer Publisher: `hello` → `MashiroTK`**
  - **F. Window/taskbar icon** — `app.setWindowIcon(QIcon(":/AutoSuisei.ico"))` + ฝัง ico เข้า `resources.qrc` (alias `../../installer/AutoSuisei.ico`); เดิม `main.cpp` ไม่ได้ setWindowIcon เลย Qt ไม่ใช้ไอคอน exe
  - bump **1.0.3** → `Setup-1.0.3.exe` + `update-package.zip` → release **v1.0.3**
- **Technical Notes:**
  - layout reshuffle = สลับลำดับ `root->addWidget/addLayout`: `addWidget(split,1)` กลายเป็น child แรก → renameRow → header block (widgets สร้างไว้ที่เดิม แค่ add ทีหลัง — locals ยังอยู่ใน scope constructor)
  - Review table **คง 7 คอลัมน์** (col5 repurpose Date→Category) → OK ยัง **col6** ไม่ต้องเลื่อน index handler; Watch ลดคอลัมน์จริง (Date หาย, kCols 5)
  - `categoryLabel` mapping duplicate ใน WatchTab + ReviewTab (5 บรรทัด — mapping product นิ่ง, เลี่ยง shared-header churn)
  - icon อ่าน .ico ผ่าน **qico plugin** (`imageformats/`) จาก qrc — deploy แล้วทั้ง dev build + bundle; ถ้าไม่ขึ้น fallback = ฝัง PNG
  - update-package **valid** (แก้ C++/installer เท่านั้น ไม่แตะ python/); **Publisher** ใน Programs and Features เปลี่ยนเฉพาะคนที่ลงใหม่ด้วย `Setup-1.0.3.exe` (อัปเดตทับผ่าน updater ไม่เขียน uninstall registry ใหม่)

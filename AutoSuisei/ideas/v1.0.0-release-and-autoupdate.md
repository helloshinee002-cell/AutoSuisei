# Idea — v1.0.0: monitor barcode + version UI + Check/Update (auto-update)

[[index]] · [[CLAUDE]] · log: [[logs/2026-06-26-v1.0.0-updater-monitor]] · ต่อจาก [[ideas/barcode-qr-serial]]

## Problem / สิ่งที่ user ขอ (ส่วนที่ 2)
1. commit รวบงาน barcode (0.9.6/0.9.7) → bump **v1.0.0** ตั้งแต่นี้
2. อ่าน **barcode/QR ของจอ Monitor** ด้วย (`Downloads/Rename/Monitor`)
3. แสดง **version ที่มุมซ้ายล่าง** ของ GUI
4. เพิ่ม **Check Version & Update Version** (auto-update)
5. **push ขึ้น GitHub** + ทำระบบ update ผ่าน **GitHub Releases + REST API** (partial "update-package", ไม่ฝัง token)

## ขอบเขต / ข้อตัดสินใจ (เคาะกับ user)
- monitor = **barcode-first CN serial** (Data Matrix + 1D Code128 ที่เอียง → rotation sweep; reformat ใส่ dash); อ่านไม่ได้ → OCR
- push/release = **ติดตั้ง gh + login** แล้ว push + `gh release create`
- update-package = **app-patch** (bundle ลบ python/ ~440MB) + **fallback full installer** ถ้า python deps เปลี่ยน
- updater ใช้ **public API ไม่ต้อง token**; apply ผ่าน `.bat` (extract→overwrite→relaunch) elevate ผ่าน UAC

## Acceptance
- GUI โชว์ `v1.0.0` ซ้ายล่าง + ปุ่ม Check; เทียบ tag_name แล้วแจ้ง update
- monitor barcode อ่าน CN ได้จริง (วัด ~60% decode, ที่เหลือ OCR fallback), ไม่ regress
- ทดสอบ E2E updater ได้เมื่อมี release จริง (Phase 2b หลัง gh login)
